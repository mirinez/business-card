### Projektbeschreibung

Dieses Projekt ist eine statische Business Card (Visitenkarte), erstellt ausschließlich mit HTML und CSS. Ziel der Aufgabe war es, die wichtigsten Grundkonzepte beider Sprachen praktisch anzuwenden: Seitenstruktur, responsive Layouts mit Flexbox, Hover-Effekte, Custom Fonts und die Trennung von Inhalt (HTML) und Design (CSS). Die Business Card präsentiert ein professionelles Profil mit Kontaktmöglichkeiten und ist vollständig auf Englisch, um mit LinkedIn, GitHub und Portfolio links.
